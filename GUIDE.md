## 🚀 快速开始

### 第一步：打开应用

1. 使用现代浏览器（推荐 Chrome、Firefox、Edge、Safari）
2. 双击打开 `index.html` 文件，或访问部署的网址
3. 等待加载完成，看到3D斗栱模型出现在画面中央

### 第二步：尝试基本操作

- **旋转视角**：按住**左键**拖动鼠标
- **平移视角**：按住**右键**拖动鼠标  
- **缩放**：滚动**鼠标滚轮**向前/向后

1. 点击控制面板的 **"💥 拆解"** 按钮
2. 观察斗栱如何分层飞散开来
3. 点击 **"🧩 组装"** 按钮，观看组装过程

## 🎮 功能详解

### 1️⃣ 选择朝代

| 朝代 | 时期 | 建筑特征 | 代表建筑 |
|------|------|----------|----------|
| **唐** | 618-907 | 斗栱雄大、出跳深远 | 佛光寺东大殿 |
| **宋** | 960-1279 | 斗栱精致、装饰性强 | 晋祠圣母殿 |
| **明清** | 1368-1912 | 斗栱缩小、斗口制 | 故宫太和殿 |

**操作**：点击朝代按钮切换，模型会根据朝代特征调整比例和样式。

### 2️⃣ 模型操作

| 按钮 | 功能 | 说明 |
|------|------|------|
| 💥 **拆解** | 播放拆解动画 | 构件分层飞散，展示内部结构 |
| 🧩 **组装** | 播放组装动画 | 构件自动归位，恢复完整形态 |
| 🔄 **重置视图** | 恢复初始状态 | 重置视角、拆解程度、材分等级 |

#### 拆解程度滑块

- **范围**：0%（完全组装）~ 100%（完全拆解）
- **用法**：拖动滑块可精确控制每个构件的位置
- **技巧**：停在50%左右可以清晰观察各层关系

### 3️⃣ 材分等级

**什么是材分制？**
> 材分制是中国古代建筑的模数制度，以"材"为基本单位。1材 = 15份，材广（高）:材厚（宽）= 3:2。

| 等级 | 名称 | 材广 | 材厚 | 用途 |
|------|------|------|------|------|
| 1等 | 一等材 | 9寸 | 6寸 | 殿身9-11间 |
| 2等 | 二等材 | 8.25寸 | 5.5寸 | 殿身5-7间 |
| 3等 | 三等材 | 7.5寸 | 5寸 | 殿身3-5间 |
| 4等 | 四等材 | 7.2寸 | 4.8寸 | 殿身3间 |
| 5等 | 五等材 | 6.6寸 | 4.4寸 | 殿身3间、厅堂5间 |
| 6等 | 六等材 | 6寸 | 4寸 | 亭榭、小厅堂 |
| 7等 | 七等材 | 5.25寸 | 3.5寸 | 亭榭、小殿 |
| 8等 | 八等材 | 4.5寸 | 3寸 | 小亭榭、藻井 |

**操作**：拖动滑块切换1-8等材，观察模型尺寸的实时变化。

### 4️⃣ 构件显示开关

可以独立控制以下构件的显示/隐藏：

- ☑️ **斗栱** - 核心构件，包括栌斗、华栱、横栱、小斗等
- ☑️ **梁架** - 承载屋顶重量的主要梁结构
- ☑️ **柱网** - 支撑建筑的柱子系统
- ☑️ **檐屋** - 屋檐和屋面结构
- ☑️ **平闇/平棊** - 天花板或吊顶
- ☑️ **台基** - 建筑的基础平台

**用法**：取消勾选可隐藏对应构件，便于观察特定部分。

## 📚 建筑知识

### 什么是斗栱？

**斗栱**（dǒu gǒng），又称**铺作**，是中国古代建筑特有的结构构件，位于柱顶与梁架之间。

**功能**：
- 🔧 **结构功能**：承托屋檐重量，将荷载传递到柱子
- 📐 **模数功能**：作为建筑尺度的基准单位
- 🎨 **装饰功能**：丰富建筑立面，体现等级制度

**组成**：
- **斗**：方形木块，像倒扣的斗
- **栱**：弓形短木，像弯曲的弓
- **昂**：斜置构件，像下垂的尾巴

### 斗栱的构件名称

```
        慢栱/令栱 ─────┬─────
                     │
        瓜子栱 ─────┼─────
                     │
        交互斗 ─────┼─────
                     │
        华栱 ───────┼──────→ 出跳方向
                     │
        栌斗 ───────┴────── 柱头
```

| 构件 | 说明 |
|------|------|
| **栌斗** | 底座大斗，坐在柱头上 |
| **华栱** | 纵向出跳的栱，垂直于墙面 |
| **泥道栱** | 横向栱，在华栱之上 |
| **瓜子栱** | 短横向栱，形如瓜子 |
| **慢栱** | 长横向栱，连接两瓜子栱 |
| **交互斗** | 小斗，在华栱跳头上 |
| **齐心斗** | 小斗，在横栱中心 |
| **散斗** | 小斗，在横栱两端 |

### 什么是材分制？

**材分制**是中国古代建筑的模数制度，最早系统记载于宋代《营造法式》。

**基本单位**：
- **材**：基本单位，1材 = 15份
- **份**：最小单位
- **栔**（qì）：辅助单位，1栔 = 6份
- **足材**：1材 + 1栔 = 21份

**比例关系**：
```
材广（高）: 材厚（宽）= 3 : 2

一等材：
- 材广 = 9寸 = 15份
- 材厚 = 6寸 = 10份
- 1份 = 6分（0.6寸）
```

**八等材**：根据建筑规模选择不同等级的"材"作为设计基准。

### 历代斗栱演变

| 时期 | 特征 | 斗栱高度/柱高比例 |
|------|------|-------------------|
| **唐代** | 斗栱雄大、出跳深远、功能性强 | 约 1:2 ~ 1:3 |
| **宋代** | 斗栱精致、装饰性增强、材分制完善 | 约 1:3 ~ 1:4 |
| **明清** | 斗栱缩小、密集排列、斗口制 | 约 1:5 ~ 1:6 |

## ❓ 常见问题解答

### Q1: 模型加载很慢/显示不出来？

**可能原因**：
- 浏览器版本过旧
- 显卡驱动问题
- 网络问题（如使用在线版本）

**解决方法**：
1. 更新浏览器到最新版本
2. 尝试使用 Chrome 或 Firefox
3. 检查显卡驱动是否最新
4. 刷新页面重试

### Q2: 拆解动画卡住或不流畅？

**解决方法**：
1. 关闭其他占用资源的程序
2. 降低浏览器缩放比例（按 Ctrl+0 恢复100%）
3. 尝试使用"拆解滑块"手动控制，代替自动动画

### Q3: 如何保存我喜欢的视角？

目前版本暂不支持保存视角，建议：
1. 记住达到该视角的操作步骤
2. 点击"重置视图"后重新调整

### Q4: 为什么明清斗栱看起来小很多？

这是历史事实！明清时期斗栱的结构功能减弱，装饰功能增强，因此尺寸变小、排列更密集。

### Q5: 材分等级和朝代选择有什么关系？

- **朝代选择**：改变斗栱的整体比例和风格
- **材分等级**：改变构件的具体尺寸

两者可以组合使用，例如：选择唐代 + 一等材，体验最雄伟的斗栱。

### Q6: 可以导出模型或截图吗？

目前版本暂不支持导出功能。如需截图，可使用系统截图工具：
- Windows: `Win + Shift + S`
- Mac: `Cmd + Shift + 4`
