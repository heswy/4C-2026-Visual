# 《木构千年》- 中国古代木结构建筑可视化

一个基于 Three.js 的交互式3D可视化应用，展示中国古代木结构建筑中的**斗栱**（铺作）结构。

## 🏛️ 项目简介

《木构千年》是一个教育性质的Web应用，通过3D可视化技术展示中国古代建筑的精妙结构：

- **斗栱模型**：完整的七铺作斗栱3D模型，包含栌斗、华栱、泥道栱、瓜子栱、慢栱、交互斗、齐心斗、散斗等构件
- **拆解动画**：可交互的拆解/组装动画，分层展示斗栱结构
- **材分制计算**：基于《营造法式》的材分制参数化系统
- **历代对比**：唐、宋、明清三代建筑特征对比

## 🚀 快速开始

### 方式一：直接打开（推荐）

1. 使用现代浏览器（Chrome、Firefox、Edge、Safari）
2. 直接双击打开 `index.html` 文件
3. 或使用本地服务器：

```bash
# 进入项目目录
cd mugou-qiannian

# 使用 Python 启动本地服务器
python -m http.server 8000

# 或使用 Node.js
npx serve .

# 然后访问 http://localhost:8000
```

### 方式二：VS Code Live Server

1. 在 VS Code 中安装 "Live Server" 扩展
2. 右键点击 `index.html` → "Open with Live Server"

## 🎮 功能说明

### 3D 交互控制

| 操作 | 说明 |
|------|------|
| 左键拖动 | 旋转视角 |
| 右键拖动 | 平移视角 |
| 滚轮 | 缩放 |
| 鼠标悬停 | 显示构件名称 |

### 控制面板功能

1. **朝代选择**：切换唐、宋、明清三代建筑风格
2. **模型操作**：
   - 拆解/组装按钮：触发动画
   - 拆解滑块：精确控制拆解程度
   - 重置视图：恢复初始状态
3. **材分等级**：调整1-8等材，实时改变模型尺寸
4. **构件显示**：开关各类构件的显示

### 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `空格` | 切换拆解/组装 |
| `R` | 重置视图 |
| `ESC` | 关闭移动端面板 |

## 🏗️ 技术栈

- **Three.js** - 3D 渲染引擎
- **原生 JavaScript** - 无框架依赖
- **CSS3** - 现代化样式
- **Canvas API** - 木纹纹理生成
